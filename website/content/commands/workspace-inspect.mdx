# Waypoint Workspace inspect

Command: `waypoint workspace inspect`

Output Workspace information.

@include "commands/workspace-inspect_desc.mdx"

## Usage

Usage: `waypoint workspace inspect [<name>]`

Output information about a waypoint workspace, including the projects and
last known activity

#### Global Options

- `-plain` - Plain output: no colors, no animation.
- `-app=<string>` - App to target. Certain commands require a single app target for Waypoint configurations with multiple apps. If you have a single app, then this can be ignored.
- `-workspace=<string>` - Workspace to operate in.
