# Waypoint

Learn and develop your knowledge of Waypoint with these tutorials and code resources.

## Getting Started

- Begin with this brief [quick start tutorial](/docs/getting-started).
- Continue with an introductory tutorial on [Docker](https://learn.hashicorp.com/collections/waypoint/get-started), [Kubernetes](https://learn.hashicorp.com/collections/waypoint/get-started-kubernetes), or [Nomad](https://learn.hashicorp.com/collections/waypoint/get-started-nomad).

## Deploy to the Cloud

Take Waypoint to the cloud with tutorials that deploy to public cloud infrastructure.

- AWS - Visit the [Elastic Container Service tutorial](https://learn.hashicorp.com/tutorials/waypoint/aws-ecs?in=waypoint/aws) or the Elastic Kubernetes Service tabs in the [Kubernetes tutorial](https://learn.hashicorp.com/collections/waypoint/get-started-kubernetes).
- Azure - Deploy to [Azure Container Instances](https://learn.hashicorp.com/tutorials/waypoint/azure-container-instance?in=waypoint/azure).
- Google Cloud - Learn from the [Google Cloud Run tutorial](https://learn.hashicorp.com/tutorials/waypoint/google-cloud-run?in=waypoint/google-cloud) or the Google Kubernetes Engine tabs in the [Kubernetes tutorial](https://learn.hashicorp.com/collections/waypoint/get-started-kubernetes).

## Explore Code Samples

A growing number of code samples is available in the [hashicorp/waypoint-examples](https://github.com/hashicorp/waypoint-examples) repository at GitHub. Use these examples or contribute your own.

```shell-session
git clone https://github.com/hashicorp/waypoint-examples.git
```

## Extend

The [plugin documentation](/plugins) describes more resources available to use in your Waypoint configuration. Contribute by writing your own plugin with this [plugin tutorial](/docs/extending-waypoint/example-plugin).

Integrate Waypoint into your continuous delivery workflow with examples for [GitHub Actions](docs/automating-execution/github-actions), [CircleCI](/docs/automating-execution/circle-ci), or [Jenkins](/docs/automating-execution/jenkins).
