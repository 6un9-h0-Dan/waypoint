# Waypoint Artifact list-builds

Command: `waypoint artifact list-builds`

Build a new versioned artifact from source.

@include "commands/artifact-list-builds_desc.mdx"

## Usage

Usage: `waypoint artifact list-builds [options]`

#### Global Options

- `-workspace=<string>` - Workspace to operate in. Defaults to 'default'.

#### Command Options

- `-workspace-all` - List builds in all workspaces for this project and application.
