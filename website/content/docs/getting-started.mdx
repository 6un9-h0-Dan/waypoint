# Get Started with Waypoint

## Deploy Your First App with Waypoint and Docker

To help introduce you to Waypoint, we'll use Docker locally to deploy a simple
application. This will teach you the basic components of Waypoint and show it working with
minimal dependencies.

By following the guide below, you will be ready to start using additional plugins
such as Kubernetes to deploy a more complex platform.
